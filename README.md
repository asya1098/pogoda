# pogoda

**pogoda - Консольное приложение для получения текущей погоды**

### установка 
- git clone https://github.com/asya1098/pogoda
- cd pogoda
- bash install.sh
- в вашей домашней дериктории должен появится скрипт по запуску программы
- bash pogoda.sh
